HFT Orders API (1.0.0)

Download OpenAPI specification:

REST API for retrieving tradable instruments and placing orders on the HFT Orders platform.

Get instruments

Returns a list of instruments that match the supplied filters.

query Parameters
search
string

Search string to match ticker or instrument name (case‑insensitive substring).

ticker
string

Exact instrument ticker symbol (e.g. AAPL).

mic
string

Market Identifier Code (MIC) that specifies the trading venue. Optional. If omitted for U.S. equities, the server assumes the default venue for the ticker (e.g. XNAS for NASDAQ‑listed stocks or XNYS for NYSE‑listed stocks).

limit
integer [ 1 .. 1000 ]

Maximum number of instruments to return (≤ 1000).

offset
integer >= 0

Offset from the beginning of the result set (pagination).

is_active
boolean

Filter by active status (true = active, false = inactive).

Responses

Place a new order

Creates a new order for the specified instrument. For U.S. equities the mic field is not required; the server infers the appropriate venue.

Authorizations:
bearerAuth
Request Body schema: application/json
required
ticker
required
string

Ticker symbol of the security to trade (e.g. AAPL).

side
required
string
Enum: "buy" "sell"
quantity
required
integer

Number of shares.

price
number <double>
comment
string

Пользовательский комментарий к заявке.
